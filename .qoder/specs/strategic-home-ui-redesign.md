## 核心改动

### 1. 布局结构重构
- **主区域 + 右侧 AI 侧栏**: `flex` 布局，主区域 `flex-1`，侧栏固定 `w-96` (384px)
- **AI Chat 从主区域移出**: 不再占据 70vh 的中心位置
- **信息密度降低**: 主区域专注业务信息，对话功能独立

### 2. AI 助手侧栏设计
```
高度: 100% (随主区域)
宽度: w-96 (384px)
背景: bg-navy-900
结构:
├─ Header (固定): 标题 + 连接状态
├─ Messages (flex-1, 滚动): 对话消息
├─ Quick Actions (固定): 快捷预案按钮
└─ Input (固定): 输入框 + 发送按钮
```

### 3. GlobalDashboard 简化
- **Boss 视图**: 3列 Grid (2行5个卡片)，替代原来1行5个
- **卡片瘦身**: 移除 secondary 指标，仅保留 primary + 健康灯 + 趋势
- **Staff 视图**: 精简为列表模式，仅显示模块名 + 核心数值

### 4. 欢迎消息处理
- **移除主区域的大型欢迎面板**: 12层嵌套的复杂结构
- **信息分流**:
  - 核心结论 → AI 增长洞察卡片
  - 战果汇总 → Dashboard KPI 卡片
  - 阻塞点 → 待拍板任务列表
- **AI 侧栏首条消息**: 简洁的欢迎语

## 间距系统规范

```
区块间距:
- gap-8 (32px): 主要区块之间
- gap-6 (24px): 次级区块之间  
- gap-4 (16px): 卡片内元素
- gap-3 (12px): 紧密元素

内边距:
- p-6 (24px): 大卡片
- p-5 (20px): 中卡片
- p-4 (16px): 小卡片

圆角:
- rounded-2xl (16px): 主容器/卡片
- rounded-xl (12px): 按钮/输入框
- rounded-lg (8px): 标签/徽章
```

## 实施步骤

### Step 1: 创建 AISidebar 组件
新建 `components/AISidebar.tsx`:
- 从 StrategicHome 提取 Chat 相关逻辑
- 固定宽度 w-96，深色背景
- Header + Messages + QuickActions + Input 结构
- 角色差异化: Boss 显示"出海增长专家"，Staff 显示"工作助手"

### Step 2: 重构 StrategicHome 布局
修改 `components/StrategicHome.tsx`:
- 移除内嵌的 Chat Terminal 代码
- 改为 flex 布局: 主区域 + AISidebar
- Boss: Dashboard → AI洞察+待拍板 → ReportGenerator
- Staff: 待办任务 → 快捷操作+进度概览

### Step 3: 简化 GlobalDashboard
修改 `components/GlobalDashboard.tsx`:
- Boss 模式: 3列 Grid，卡片精简
- Staff 模式: 添加 `compact` prop，显示为简洁列表
- 移除过度动画 (双 pulse+ping)

### Step 4: 移除 Welcome Message
从 StrategicHome 删除:
- 大型欢迎面板的 JSX 代码
- 相关的 useEffect 初始化消息逻辑
- 将核心信息分流到各组件

### Step 5: 调整 Right Sidebar 内容
- 移除原有的 Stats 卡片 (信息已在 Dashboard)
- 移除 AI Insight 卡片 (移到主区域)
- Staff 的 Upload 入口移到快捷操作区

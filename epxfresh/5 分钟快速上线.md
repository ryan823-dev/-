# EPXFresh 5 分钟快速上线指南

## 🎯 目标
**5 分钟内将网站部署到互联网，可公开访问**

## ⚡ 最快方案：Netlify（推荐）

### 为什么选择 Netlify
- ✅ **完全免费** - 个人项目免费使用
- ✅ **全球 CDN** - 访问速度快
- ✅ **自动 HTTPS** - 无需配置
- ✅ **持续部署** - 连接 GitHub 自动更新
- ✅ **无需备案** - 直接使用

### 步骤（5 分钟完成）

#### 1️⃣ 准备文件（30 秒）

```bash
cd /Users/oceanlink/Documents/Qoder-1/epxfresh

# 构建静态文件
npm run build

# 生成 out/ 目录
ls out/
```

你会看到：
- index.html
- about.html
- shop.html
- 等页面...

#### 2️⃣ 注册 Netlify（1 分钟）

1. 访问：https://www.netlify.com
2. 点击 **Sign Up**
3. 使用 GitHub 账号登录（推荐）或邮箱注册
4. 完成注册

#### 3️⃣ 部署网站（2 分钟）

**方法 A: 拖拽部署（最简单）**

1. 登录 Netlify
2. 点击 **Add new site** → **Deploy manually**
3. 将 `out/` 文件夹拖拽到上传区域
4. 等待上传完成（约 30 秒）
5. 完成！获得访问地址

**方法 B: 命令行部署（更专业）**

```bash
# 安装 Netlify CLI
npm install -g netlify-cli

# 登录
netlify login

# 部署
netlify deploy --prod --dir=out
```

#### 4️⃣ 获取访问地址（30 秒）

部署完成后，你会看到：
- **临时域名**: `https://epxfresh-[随机].netlify.app`
- 点击即可访问

#### 5️⃣ 配置自定义域名（可选，2 分钟）

**如果你有域名**:

1. Netlify Dashboard → **Domain settings**
2. 点击 **Add custom domain**
3. 输入你的域名（如 `www.epxfresh.com`）
4. 在域名服务商添加 DNS 记录：
   ```
   类型：CNAME
   主机记录：www
   记录值：[你的 netlify 域名].netlify.app
   ```
5. 等待 DNS 传播（5-30 分钟）

## ⚙️ 配置环境变量（部署后）

### AI 助理配置

如果 AI 助理不可用，需要配置环境变量：

**Netlify**:
1. Dashboard → **Site settings**
2. **Environment variables**
3. 添加：
   ```
   OPENAI_API_KEY=sk-你的密钥
   NEXT_PUBLIC_SANITY_PROJECT_ID=你的 Sanity ID
   ```
4. 重新部署

**Vercel**:
1. Dashboard → **Settings** → **Environment Variables**
2. 添加相同变量
3. 重新部署

## 🆘 常见问题

### Q: 部署后页面空白？
A: 检查浏览器控制台（F12），查看错误信息

### Q: 图片不显示？
A: 确保图片使用绝对路径或 CDN 地址

### Q: AI 助理不工作？
A: 需要配置 OPENAI_API_KEY 环境变量

### Q: 访问速度慢？
A: 使用 Cloudflare CDN 或选择离用户近的部署区域

### Q: 如何更新网站？
A: 重新运行部署命令或拖拽新文件到 Netlify

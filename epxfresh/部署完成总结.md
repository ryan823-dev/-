### 自定义域名（可选）

**配置步骤**:

1. 在部署平台添加域名
2. 在域名服务商添加 DNS 记录
3. 等待 DNS 传播（5-30 分钟）

**DNS 配置示例**:
```
类型：CNAME
主机记录：www
记录值：你的域名.netlify.app 或 cname.vercel-dns.com
```

## 🔧 配置环境变量（AI 功能）

### 必须配置的变量

```bash

NEXT_PUBLIC_SANITY_PROJECT_ID=你的 Sanity Project ID
NEXT_PUBLIC_SANITY_DATASET=production
```

### 配置位置

**Netlify**:
- Dashboard → Site settings → Environment variables

**Vercel**:
- Dashboard → Settings → Environment Variables

**阿里云**:
- 需要单独部署 API 服务
